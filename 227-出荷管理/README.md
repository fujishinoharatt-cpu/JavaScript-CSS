# 227-出荷管理

## 概要

Kintone **227番アプリ（出荷管理）** 向けの一括操作スクリプト群です。

特定条件のレコードを一括更新・一括削除するためのJavaScriptカスタマイズと設定ファイルで構成されています。

---

## ファイル構成

```
227-出荷管理/
├── bulk_update.js            # 条件指定一括更新（設定ファイル方式）
├── bulk_update_config.js     # bulk_update.js 用の設定ファイル
├── bulk_update_all.js        # 全件一括更新
├── bulk_update_all_config.js # bulk_update_all.js 用の設定ファイル
├── bulk_delete.js.js         # 一括削除
├── bulk_delete_config.js.js  # 一括削除用の設定ファイル
└── CSV/                      # CSVデータ保存フォルダ
```

---

## 主な機能

### `bulk_update.js`（条件指定一括更新）
- 一覧画面に「一括更新」ボタンを追加
- `config.js` で指定した**条件に合うレコードのみ**を対象に特定フィールドを一括更新
- 100件ずつ再帰処理で全件対応（件数制限なし）
- 既に更新済みのレコードは自動スキップして**無限ループを防止**

### `bulk_update_all.js`（全件一括更新）
- 全レコードに対して一括更新を実行

### `bulk_delete.js`（一括削除）
- 条件指定による一括削除
