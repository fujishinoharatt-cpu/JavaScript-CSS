# 153-mobile-thumbnail

## 概要

Kintone **153番アプリ（工程内不適合）** のモバイル一覧画面に、添付画像のサムネイルを表示するJavaScriptカスタマイズです。

---

## ファイル構成

```
153-mobile-thumbnail/
├── mobile-thumbnail.js  # モバイル一覧画面サムネイル表示
└── custom-gallery.js    # カスタムギャラリー表示
```

---

## 主な機能

### `mobile-thumbnail.js`
- **モバイル一覧画面**（`mobile.app.record.index.show`）に対応
- 各レコードの「不適合画像」フィールドから画像を取得し60×60pxのサムネイルとして表示
- KintoneのスクロールによるDOM再構築に対応するため**1秒間隔で再描画チェック**を実施
- ラベル幅35% / 値幅65% でレイアウトを最適化するCSSを動的注入
- サムネイルクリックで画像を別タブで拡大表示

### `custom-gallery.js`
- PC・モバイル向けのカスタムギャラリー表示
