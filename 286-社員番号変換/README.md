# 286-社員番号変換

## 概要

Kintone **286番アプリ（社員情報管理）** の詳細画面に、**LINEチャット統合ビュー**を表示するJavaScriptカスタマイズです。

社員のLINEユーザーIDをもとに、チャット履歴・メッセージ送信・定期配信予約の3機能を1画面で操作できます。

---

## ファイル構成

```
286-社員番号変換/
├── 286-chat-view.js   # チャット統合ビュー（メイン）
├── 286-chat-view.css  # チャットUI用CSS
└── update_names.js    # 社員名更新スクリプト
```

---

## 主な機能（286-chat-view.js）

### 3カラムのチャット統合UI（詳細画面）

| カラム | アプリ | 機能 |
|---|---|---|
| 左（チャット履歴） | App 298 | LINEの送受信メッセージ履歴を時系列表示 |
| 中央（メッセージ送信） | App 299 | テキスト・画像をLINEユーザーへ送信 |
| 右（定期配信予約） | App 300 | 定期配信の登録・編集・ステータス管理 |

### チャット履歴表示（App 298）
- USER（受信）は左、BOT（送信）は右のLINE風バブルで表示
- 日付区切りを自動挿入
- 添付画像はXHR経由で取得してインライン表示

### メッセージ送信（App 299）
- テキスト入力・画像添付（Canvas圧縮でリサイズ後アップロード）
- 送信後は履歴を自動リロード

### 定期配信予約（App 300）
- 配信予定日・サイクル（1回/毎日/毎週/毎月/毎年）・ステータスを設定して登録
- 登録済み予約の編集・キャンセルに対応
