# 303-mobile-thumbnail

## 概要

Kintone **303番アプリ（工程内不適合発見 v2）** のモバイル一覧画面に、添付画像のサムネイルを表示するJavaScriptカスタマイズです。

153番アプリと同じ構造で、v2アプリ向けに調整されたバージョンです。

---

## ファイル構成

```
303-mobile-thumbnail/
├── mobile-thumbnail.js  # モバイル一覧画面サムネイル表示
└── custom-gallery.js    # カスタムギャラリー表示
```

---

## 主な機能

- **モバイル一覧画面**（`mobile.app.record.index.show`）に対応
- 各レコードの添付画像フィールドから画像をXHR経由で取得し60×60pxのサムネイルを表示
- スクロールによるDOM再構築に対応するため**1秒間隔で再描画チェック**を実施
- サムネイルクリックで画像を別タブで拡大表示
