# 231-運送便管理

## 概要

Kintone **231番アプリ（運送便管理）** 向けのJavaScriptカスタマイズです。

**227番アプリ（出荷管理）** と連携し、出荷日ごとの才数集計や不足レコードの自動補完、CSV出力などの機能を提供します。

---

## 主な機能

### 「➕ 不足分を追加」ボタン（一覧画面）
App 227（出荷管理）にあってApp 231（運送便管理）にない**出荷日のレコードを自動で補完**します。

1. App 231の既存出荷日を全取得
2. App 227の全レコードから出荷日を取得
3. 不足分（差分）のみを特定して一括POST

### 「🔄 一括再集計」ボタン（一覧画面）
表示中のレコードの集計結果をApp 227から再取得して更新します。
- 冨士才数・オリジン才数・合計才数を集計してApp 231に書き戻し

### 「📊 期間指定CSV出力」ボタン（一覧画面）
指定期間のレコードをCSVファイルとしてダウンロードします。
- BOM付きUTF-8でExcel対応

### 「🔄 レコード別集計」ボタン（詳細・編集画面）
1レコード単位でApp 227からその出荷日の才数を集計して画面に反映します（保存前確認可能）。
