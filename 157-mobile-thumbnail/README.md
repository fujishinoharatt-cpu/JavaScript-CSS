# 157-mobile-thumbnail

## 概要

Kintone **157番アプリ（不安全発見）** のモバイル一覧画面に、添付画像のサムネイルを表示するJavaScriptカスタマイズです。

---

## ファイル構成

```
157-mobile-thumbnail/
└── custom-gallery-157.js  # サムネイル表示・カスタムギャラリー
```

---

## 主な機能

- **モバイル一覧画面**に対応した画像サムネイル表示
- 不安全発見の現場写真を一覧でプレビュー可能
- Kintone File APIを使用したXHR経由での画像取得（認証ヘッダー付き）

---

| アプリID | アプリ名 |
|---|---|
| 157 | 不安全発見 |
