# 314-管理アプリ

## 概要

Kintone上の**アプリ管理情報（314番アプリ）**を操作するためのJavaScriptカスタマイズです。

Kintone内に存在する全アプリを自動取得し、未登録のアプリを一括で314番管理アプリに登録するボタンを一覧画面に追加します。

---

## 主な機能

### 「kintone内の全アプリを取得・登録」ボタン

一覧画面のヘッダーに表示されるボタンで、以下の処理を実行します：

1. `/k/v1/apps.json` でKintone内の全アプリ情報を取得（最大100件）
2. 314番アプリに既に登録されているアプリIDを取得
3. **未登録のアプリのみ**を差分チェックで抽出
4. 確認ダイアログを表示して承認後、一括登録（POST）

---

## 登録されるフィールド

| フィールドコード | 内容 |
|---|---|
| `appId` | アプリID |
| `appName` | アプリ名 |
| `appUrl` | アプリの一覧画面URL |

---

## ファイル構成

```
314-管理アプリ/
└── sync-apps.js    # Kintoneカスタマイズ用JS（一覧画面ボタン追加）
```

---

## 使い方

1. `sync-apps.js` をKintone 314番アプリのJavaScriptカスタマイズとして登録
2. 314番アプリの一覧画面を開く
3. 「**kintone内の全アプリを取得・登録**」ボタンをクリック
4. 未登録アプリの件数が表示されるので確認・承認する

> **注意**: アプリ一覧の取得には適切な管理者権限が必要です。
