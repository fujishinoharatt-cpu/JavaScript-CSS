# 167-mobile-thumbnail

## 概要

Kintone **167番アプリ（不適合管理）** のモバイル一覧画面に、添付画像のサムネイルを表示するJavaScriptカスタマイズです。

---

## ファイル構成

```
167-mobile-thumbnail/
└── custom-gallery-167.js  # サムネイル表示・カスタムギャラリー
```

---

## 主な機能

- **モバイル一覧画面**に対応した画像サムネイル表示
- 不適合管理の添付写真を一覧でプレビュー可能
- Kintone File APIを使用したXHR経由での画像取得（認証ヘッダー付き）

---

## 設置先アプリ

| アプリID | アプリ名 |
|---|---|
| 167 | 不適合管理 |
