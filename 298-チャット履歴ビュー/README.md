# 298-チャット履歴ビュー

## 概要

Kintone **298番アプリ（チャット履歴）** の一覧画面に、**LINEチャット形式**でメッセージ履歴を表示するJavaScriptカスタマイズです。

全件を取得してLINEアプリに似たUI（吹き出し形式）で表示し、ユーザー単位のフィルタリングが可能です。

---

## ファイル構成

```
298-チャット履歴ビュー/
├── 298-chat-view.js    # 298番アプリ専用チャットビュー（メイン）
├── 298-chat-view.css   # チャットUI用CSS（298専用）
├── chat-view.js        # 汎用チャットビュー
├── chat-view.css       # 汎用チャットビュー用CSS
└── walkthrough.html    # 使い方ガイドページ
```

---

## 主な機能（298-chat-view.js）

### チャット形式の一覧表示
- 全レコードを500件ずつ再帰取得して全件表示
- タイムスタンプ順にソート（同時刻はレコードID順）
- USER（受信）は左、BOT（送信）は右のLINE風バブルで表示
- 日付が変わるたびに日付区切り線を自動挿入

### ユーザーフィルタリング
- ドロップダウンでLINEユーザーを選択し、特定ユーザーのメッセージのみを表示
- ユーザー名・LINEIDを自動抽出してセレクトボックスに一覧表示

### 画像表示
- 添付ファイルはKintone File APIをXHR経由（`X-Requested-With`ヘッダー付き）でBlobとして取得
- クリックでダウンロード可能

### 件数表示
- 総メッセージ数・検出ユーザー数をヘッダーに表示

## 使い方

1. `298-chat-view.js` と `298-chat-view.css` をApp 298のJavaScript/CSSカスタマイズとして登録
2. 一覧ビューのカスタムHTMLに `<div id="chat-view-root"></div>` を配置
3. 一覧画面を開くとチャット形式でメッセージが表示される
